Font Weight

Examples:

Thin text - 100
Extra Light - 200
Light text - 300
Normal text - 400
Medium text - 500
Semi Bold - 600
Bold text - 700
Extra Bold - 800
Black text - 900
Bold (keyword)
Bolder (keyword)

CSS Code:

.thin-text {
  font-weight: 100;
}

.light-text {
  font-weight: 300;
}

.normal-text {
  font-weight: 400;
}

.bold-text {
  font-weight: 700;
}

.bolder-text {
  font-weight: 900;
}

.keyword-bold {
  font-weight: bold;
  /* Same as 700 */
}

.keyword-bolder {
  font-weight: bolder;
  /* Bolder than parent */
}

Font Family

Examples:

Serif Font - Georgia, Times New Roman
Sans-serif Font - Arial, Helvetica
Monospace Font - Courier New, Monaco
Cursive Font - Brush Script MT

CSS Code:

.serif-font {
  font-family: Georgia, Times New Roman, serif;
}

.sans-font {
  font-family: Arial, Helvetica, sans-serif;
}

.mono-font {
  font-family: 'Courier New', Monaco, monospace;
}

Text Transform

Examples:

none - original text
capitalize - first letter of each word
uppercase - all capital letters
LOWERCASE - ALL SMALL LETTERS

CSS Code:

  text-transform: capitalize;
}

.uppercase {
  text-transform: uppercase;
}

.lowercase {
  text-transform: lowercase;
}
